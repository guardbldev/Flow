What is Flow?

Flow is a single-language solution for UI, app logic, and state. No HTML/CSS/JS split. One syntax, beautiful defaults, advanced extensibility.

Beginner friendly.
Visual-first, but not limiting.
Opinionated yet overridable.

Features

  • Unified layout, logic & state
  • Automatic, beautiful layouts
  • Reactive state and bindings
  • Advanced events & animations
  • Platform overrides for web/desktop/mobile
  • Full WASM/native output support
  • Simple but deep

Quick Examples

screen Home:
    title "Welcome!"

    state clicks = 0

    button "Click Me":
        clicks += 1

    text "You clicked {clicks} times."
