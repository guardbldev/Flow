Getting Started

Install Flow

  • Clone the repo: git clone https://github.com/example/flow-lang.git
  • Build with CMake:
cd flow
cmake .
make

Your First Flow App

screen App:
    title "My First Flow App"
    button "Press Me":
        show message "Hello from Flow!"

Run

./flow example.flow

See more examples!
